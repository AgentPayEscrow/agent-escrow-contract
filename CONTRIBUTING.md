# Contributing to Agent Escrow Frontend

## How to Contribute

1. Fork the repository
2. Create a branch: `git checkout -b feature/your-feature`
3. Make changes to `index.html`
4. Test by opening the file in your browser
5. Take a screenshot of your changes
6. Commit: `git commit -m "feat: description (Closes #123)"`
7. Push: `git push origin feature/your-feature`
8. Open a Pull Request with screenshot

## Issue Points (Drips Wave)

| Difficulty | Points | Time |
|------------|--------|------|
| Easy | 100 | 2-4 hours |
| Medium | 150 | 4-8 hours |
| Hard | 200 | 8-16 hours |

## Pull Request Requirements

- Title format: `type: description (points) (Closes #issue)`
- Description must include `Closes #issue_number`
- Screenshot required for UI changes
